Please follow the below instruction to run this demo.
- Extract the zip file (classdemo.zip) 
- Open a new terminal window 
- Type the following commands
  1 - cd classdemo 
  2 - npm install
  3 - node app 
the second command will install all the packages used in this demo
the third will run the server. 
To check if the server is actually running, open a browser and go to the following link
localhost:3333
